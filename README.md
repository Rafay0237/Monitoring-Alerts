# Alerts Monitor

**Alerts Monitor** is an open-source alerts monitoring system designed for developers to track exceptions and crashes in their applications in real-time. It provides a **frontend dashboard** and a **backend API**, making it easy to monitor errors, set thresholds, and receive email notifications when those thresholds are exceeded.  

---

## Features

- **User Authentication:** Secure sign up and login.  
- **Project Management:** Create and manage multiple projects.  
- **Error Thresholds:** Set maximum allowed exceptions or crashes per project.  
- **Email Alerts:** Receive email notifications automatically when errors exceed limits.  
- **API Key Integration:** Generate and copy project-specific API keys for client-side error tracking.  
- **Open-Source API:** Built using **Node.js** and **MongoDB**, easy to extend and customize.
